👤 User.role values

admin employee

Admins and employees share the same model. Role is stored as a plain string with a default of employee.

🔒 Security Notes

password — bcrypt hashed, cost 12. Never stored in plain text.

encryptedText — AES-256-CBC encrypted before saving. Raw content never persists in MongoDB.

filePath — stored in /uploads, never publicly accessible.

🔗 Relationships

User → Secrets
